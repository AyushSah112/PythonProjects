# Expense Tracker

A simple Python application for recording and managing daily expenses. It helps organize expense records and provides a beginner-friendly introduction to data management.

## Features

- Add expenses
- View expense records
- Simple and easy-to-use interface
- Clean and readable code

## How to Run

```bash
python main.py
```

## Project Structure

```
Expense-Tracker/
├── main.py
└── README.md
```

## Purpose

This project was created to practice file handling, dictionaries, user input, and Python programming concepts.
