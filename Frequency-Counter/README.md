# Frequency Counter

A Python program that counts the frequency of words or characters in a given input. This project demonstrates basic text processing techniques.

## Features

- Count word frequency
- Count character frequency
- Clean and readable code
- Beginner-friendly implementation


## How to Run

```bash
python main.py
```

## Project Structure

```
Frequency-Counter/
├── main.py
└── README.md
```

## Purpose

This project was created to practice dictionaries, loops, string manipulation, and problem-solving in Python.
