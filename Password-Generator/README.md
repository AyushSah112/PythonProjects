# Password Generator

A Python application that generates strong and random passwords based on user-defined requirements.

## Features

- Random password generation
- Custom password length
- Secure character selection
- Beginner-friendly implementation


## How to Run

```bash
python main.py
```

## Project Structure

```
Password-Generator/
├── main.py
└── README.md
```

## Purpose

This project was created to practice the `random` module, strings, loops, and user input in Python.
