🔐 Password Strength Checker

A beginner-friendly Python project that evaluates the strength of a password using a simple scoring system.

Features

- Detects lowercase letters
- Detects uppercase letters
- Detects numbers
- Detects special symbols
- Checks password length
- Calculates a score based on password complexity
- Classifies the password as:
  - Weak
  - Medium
  - Strong
  - Very Strong

How It Works

The program checks whether the password contains:

- At least one lowercase letter
- At least one uppercase letter
- At least one number
- At least one special symbol

It also gives additional points based on the password length. The total score determines the password strength.

How to Run

1. Clone or download this repository.
2. Open the project in any Python IDE or terminal.
3. Run the Python file.
4. Enter a password.
5. The program will analyze it and display its strength.
