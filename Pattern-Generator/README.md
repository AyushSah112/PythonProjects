# Pattern Generator

A collection of Python programs that generate different star, number, and character patterns using loops and conditional statements.

## Features

- Star patterns
- Number patterns
- Character patterns
- Beginner-friendly examples


## How to Run

```bash
python main.py
```

## Project Structure

```
Pattern-Generator/
├── main.py
└── README.md
```

## Purpose

This project was created to strengthen logic-building skills and practice loops, nested loops, and pattern-based programming in Python.
