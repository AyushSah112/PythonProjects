# QR Code Generator

A simple Python application that generates QR codes from user-provided text or URLs.

## Features

- Generate QR codes
- Save QR code as an image
- Easy-to-use interface
- Beginner-friendly implementation


## How to Run

```bash
python main.py
```

## Project Structure

```
QR-Code-Generator/
├── main.py
└── README.md
```

## Purpose

This project was created to practice working with external Python libraries and generating QR code images.
