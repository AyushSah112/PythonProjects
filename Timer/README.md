# Timer

A simple countdown timer built with Python that counts down from a user-defined duration and notifies the user when the time is up.

## Features

- Countdown timer
- User-defined duration
- Simple command-line interface
- Beginner-friendly implementation


## How to Run

```bash
python main.py
```

## Project Structure

```
Timer/
├── main.py
└── README.md
```

## Purpose

This project was created to practice loops, time handling, user input, and basic Python programming concepts.
