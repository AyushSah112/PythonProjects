To-Do List

A simple command-line To-Do List application developed in Python. This project helps users organize their daily tasks by allowing them to add, view, complete, delete, and clear tasks. All tasks are stored in a text file, so they remain available even after the program is closed.

Features

- View all saved tasks
- Add one or multiple tasks
- Mark tasks as completed
- Delete a selected task
- Clear all tasks
- Save tasks permanently using a text file
- Handle invalid user input with exception handling

How It Works

The program displays a menu with different options. Based on the user's choice, it performs the requested operation.

- View Tasks – Displays all saved tasks with numbering.
- Add Task – Adds one or more new tasks to the list.
- Mark Task as Completed – Marks a selected task as completed.
- Delete Task – Removes a selected task from the list.
- Clear All Tasks – Deletes every task from the file.
- Exit – Closes the application.

All task data is stored in "to-do-list.txt", allowing tasks to persist between program runs.
